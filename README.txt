Expense Tracker Bot Setup Instructions:

1. Install Python 3.10 or newer.
2. Open Terminal inside the project folder.
3. Install libraries:
   pip install -r requirements.txt
4. Open 'expense_bot.py' and replace 'YOUR_DISCORD_BOT_TOKEN' with your actual Discord bot token.
5. Run the bot:
   python expense_bot.py
6. In your Discord server, you can use commands:
   !add amount description
   !balance
   !report
   !pdf_report
